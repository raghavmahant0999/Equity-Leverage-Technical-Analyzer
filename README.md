# Equity-Leverage-Technical-Analyzer
A Python-based quantitative tool that synthesizes corporate fundamental health (WACC, Debt-to-Equity, ICR) with technical momentum indicators (50/200 SMA Crossovers, RSI) to evaluate asset performance.
A quantitative analysis tool designed to bridge the gap between corporate financial health and market price action. By integrating fundamental metrics (WACC, Debt-to-Equity) with technical sentiment (Golden Crossovers, RSI), this tool provides a multi-dimensional perspective on asset risk and momentum.
